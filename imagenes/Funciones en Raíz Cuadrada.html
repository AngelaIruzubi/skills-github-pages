El switch es útil para tomar decisiones en base a diferentes valores de una variable, y podríamos emplearlo para manejar errores o tipos de resultados (por ejemplo, cuando el discriminante es negativo, cero o positivo).

        float b = pedirCoeficiente("b");

        float discriminante = calcularDiscriminante(a, b, c);
        int caso = determinarCaso(discriminante, a);

        mostrarResultado(caso, a, b, discriminante);
    }

    // Función para pedir un coeficiente
    public static float pedirCoeficiente(String coeficiente) {

        return scanner.nextFloat();

    // Función para calcular el discriminante
    public static float calcularDiscriminante(float a, float b, float c) {
        return b * b - 4 * a * c;

    public static int determinarCaso(float discriminante, float a) {
        if (discriminante < 0) {
            return 1;  // Caso: no hay raíces reales
        } else if (2 * a == 0) {

        } else {

        }

    public static void mostrarResultado(int caso, float a, float b, float discriminante) {

                System.out.println("No existen números reales.");
                break;
            case 2:
                System.out.println("No se puede dividir entre 0.");

                float x2 = (-b - raiz) / denominador;

            default:

        }
    }

    // Función para contar monedas
    public static double contarMonedas(double cantidad, int valor) {
        int numMonedas = (int) (cantidad / valor); // Número de monedas de la denominación
        if (numMonedas > 0) {
            System.out.println(numMonedas + " Moneda(s) de " + (valor >= 100 ? (valor / 100.0) : valor) + " euros");

Observaciones

  • Al final, la cantidad se reduce a cero después de pasar por todas las denominaciones.

Este enfoque permite que el código sea modular y más fácil de mantener, ya que cada función realiza una tarea específica.

java
import java.util.Scanner;

public class DesgloseEuros {

    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        // Cadena con las denominaciones de billetes y monedas en orden descendente
        String denominaciones = "500 200 100 50 20 10 5 2 1 0.5 0.2 0.1 0.05 0.02 0.01";

        System.out.print("Introduce la cantidad en euros: ");
        double cantidad = scanner.nextDouble();

        // Creamos un Scanner adicional para leer las denominaciones desde el String
        Scanner denominacionScanner = new Scanner(denominaciones);

        // Procesar cada denominación utilizando hasNext() y nextDouble()
        while (denominacionScanner.hasNext()) {
            // Leer la próxima denominación como double
            double valor = denominacionScanner.nextDouble();

            // Calcular cuántos billetes o monedas de la denominación actual se pueden usar
            int cuenta = (int) (cantidad / valor);

            // Si se puede usar al menos un billete o moneda de esta denominación, lo mostramos
            if (cuenta > 0) {
                // Determinar si es billete o moneda para mostrar el mensaje adecuado
                if (valor >= 5) {
                    System.out.println(cuenta + " Billete(s) de " + valor + " euros");
                } else {
                    System.out.println(cuenta + " Moneda(s) de " + valor + " euros");
                }
                // Actualizar la cantidad restante
                cantidad -= cuenta * valor;
            }
        }
        
        denominacionScanner.close();
        scanner.close();
    }
}

Explicación del Código

  1. Cadena de Denominaciones: Se usa una cadena (String) que contiene las denominaciones de billetes y monedas en orden descendente, separadas por espacios.
  2. Scanner para Procesar la Cadena: Se crea un segundo Scanner, llamado denominacionScanner, que procesa la cadena de denominaciones utilizando hasNext() para saber si hay más elementos y nextDouble() para obtener cada valor.
  3. Cálculo del Desglose:
    • Para cada valor, calcula cuántos billetes o monedas se pueden usar (cuenta = (int) (cantidad / valor)).

package ejerciciosExtra;

import java.util.Scanner;
import java.util.regex.Pattern;

public class ExpresionesReg {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        
        // Expresión regular para un código postal de cinco dígitos donde el primer dígito no es cero
        String regexp = "^[1-9][0-9]{4}$";

        // Pedir al usuario que introduzca el código postal

        // Validar el código postal

            System.out.println("El código postal es válido.");
        } else {
            System.out.println("El código postal no es válido.");
        }

        scanner.close();
    }
